# Add, Commit and Push
## Adding a file or Some changes to local git 
* Adding 1 file
    >__git add [file]("FileName.ext")__
* Adding more files 
    >__git add [file1]("fileName.ext") [file2]("fileName2.ext")__
* Add Everything
    >__git add .__
